# orq.ai API Examples

Generated by Bartolo `orq` quickstart examples.

## Setup

```sh
orq --json doctor
```

## Explore

```sh
orq agents --help
```

## Copy-Paste Commands

### Check setup

```sh
orq --json doctor
```

### Inspect server defaults

```sh
orq server list
```

### Persist the default output format

```sh
orq default-format json
```

### Explore a command group

```sh
orq agents --help
```

### Run a grouped command

```sh
orq agents create
```

### Use the raw escape hatch

```sh
orq request get /v2/agents
```
